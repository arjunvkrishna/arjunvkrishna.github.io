01. About Me

Hello! I'm Arjun. As a DevOps engineer, my job is to automate things so that when something breaks, we can confidently blame the code instead of the infrastructure. My passion is writing scripts to solve problems that people didn't even know they had yet.

I've navigated the complex backends of diverse industries like telecom, banking, gaming, food delivery, and even healthcare. It turns out that whether you're processing financial transactions or delivering a late-night pizza, everyone's servers need a babysitter. I've been that babysitter, armed with Kubernetes, GitOps, and a healthy dose of skepticism for any pull request that has more than 50 lines.

I thrive on making complex systems look easy, mostly by wrestling with YAML files and cloud consoles until they submit. Whether it's on AWS or GCP, my goal is to build things that are so automated, they practically run themselves. After all, less manual work means more time to contemplate why the build failed this time.
